# AO-scheman sparas här som JSON-filer

Varje fil representerar ett AO-blad (ett fartyg/en roll) och namnges
baserat på fartygsnamnet, t.ex.:

  ms-namdo.json
  waxholm-ii.json

Filerna skapas automatiskt via `POST /api/ao/upload`.
Mappen ska inte checkas in med sina JSON-filer (lägg till i .gitignore om det önskas).
